# 📈 Engenharia de Produção com Power BI e IA

## 📌 Visão Geral
Este projeto foi desenvolvido focado na análise de dados industriais, utilizando recursos de Inteligência Artificial do Power BI para prever a produção ao longo do tempo e detectar anomalias no processo produtivo.

## 🖼️ Visualização do Dashboard
![Capa do Dashboard](dashboard.jfif)

## 🛠️ Recursos Utilizados
* **Time Series Forecasting:** Utilização de modelos preditivos para projetar a produção com um horizonte de 5 anos.
* **Detecção de Anomalias:** Identificação de desvios significativos nos dados históricos de produção.
* **Análise Multidimensional:** Gráficos de áreas segmentados por faixa etária dos funcionários e turnos de trabalho.

## 📊 Insights de Negócio
* Identificação de tendências de queda na produção a partir de 2021.
* Monitoramento de anomalias que podem indicar falhas operacionais ou necessidade de manutenção.
* Projeção estratégica para planejamento de demanda futura.
